# social-robotics24

## assignment 3

This program is intended for Alpha Mini. 
All codes for the project is in assignment3.py and utils.py. 
Make sure to install the necessary packages. 

To run the program, run assignment3.py.

The code implements three emotional expressions in response to input cards. There are twelve input cards for 12 emotions. 
The cards may be printed out or shown from a mobile device. 
The cards can be found in cards/aruco.

To end the connection you can use a keyboard interruption or stop the program via your IDE UI.
